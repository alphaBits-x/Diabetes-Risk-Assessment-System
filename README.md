# Diabetes-Risk-Assessment-System
A fuzzy logic-based diabetes risk assessment system that calculates the probability of diabetes based on blood sugar levels and BMI. Built using Python, scikit-fuzzy, and Matplotlib.

This project implements a **Fuzzy Logic-based Diabetes Risk Assessment System** using **Python** and **scikit-fuzzy**.  
It estimates the **risk of diabetes** based on:  
✔ **Blood Sugar Levels (mg/dL)**  
✔ **Body Mass Index (BMI)**  

## 🚀 Features  
- Uses **fuzzy logic** to determine diabetes risk.  
- Implements **Triangular Membership Functions** for blood sugar, BMI, and diabetes risk.  
- Defines **9 fuzzy rules** to assess risk levels.  
- **Plots membership functions** for blood sugar, BMI, and diabetes risk.  

## 🛠️ Technologies Used  
- **Python**  
- **NumPy**  
- **Matplotlib**  
- **scikit-fuzzy (skfuzzy)**
